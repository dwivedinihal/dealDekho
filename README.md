DealDekho is a smart price-tracking app that helps users find the best deals on products across multiple online stores. It tracks price changes and product availability in real time and notifies users when prices drop or items are back in stock—saving both time and money.

🚀 Live Demo
https://deal-dekho-virid.vercel.app/

Key Features

Price Tracking: Automatically monitors product prices across various platforms

Instant Notifications: Alerts users when prices are lowest or products are back in stock

Wishlist: Save products and track price drops easily

Historical Price Data: View past price trends to make better buying decisions

User-Friendly Interface: Simple, clean, and easy to use
